# Tech Clubs Portal - Информационный портал технических кружков техникума

## 📋 О приложении

**Tech Clubs Portal** - это современное веб-приложение для студентов техникума, предоставляющее единую платформу для поиска, изучения и записи в технические кружки и секции. Приложение помогает студентам находить единомышленников, осваивать востребованные навыки и участвовать в проектах.

### 🎯 Основные функции

1. **Главная страница** - приветствие, статистика, преимущества участия
2. **Каталог кружков** - фильтрация по категориям, поиск, карточки кружков
3. **Детальная страница кружка** - полная информация, расписание, контакты куратора
4. **Форма регистрации** - подача заявки на участие в кружке
5. **Адаптивный дизайн** - работает на всех устройствах

### 🏆 Доступные кружки

- **DevForce** - Секция веб-разработки (React, Node.js)
- **CyberShield** - Кибербезопасность и этичный хакинг
- **PixelArt** - Студия UI/UX дизайна (Figma, Adobe CC)
- **Секция армрестлинга** - спортивная подготовка
- **Секция настольного тенниса** - спортивные занятия

## 🛠 Технологический стек

### Frontend
- **React 18** - библиотека для построения пользовательских интерфейсов
- **TypeScript** - статическая типизация для надежности кода
- **Vite** - современный сборщик и dev-сервер
- **TanStack Router** - клиентская маршрутизация
- **TanStack Query** - управление состоянием и кэширование
- **React Hook Form + Zod** - валидация форм
- **Framer Motion** - анимации и переходы

### UI/Стилизация
- **Tailwind CSS** - утилитарный CSS-фреймворк
- **@blinkdotnew/ui** - компонентная библиотека
- **Lucide React** - иконки
- **PostCSS + Autoprefixer** - обработка CSS

### Инструменты разработки
- **ESLint** - линтинг JavaScript/TypeScript
- **Stylelint** - линтинг CSS
- **TypeScript Compiler** - проверка типов
- **Custom CSS variable detection** - автоматическая проверка CSS-переменных

## 🚀 Запуск проекта

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```
Приложение будет доступно по адресу: `http://localhost:3000`

### Сборка для production
```bash
npm run build
```

### Предпросмотр собранной версии
```bash
npm run preview
```

## 🧪 Линтинг и проверка

### Полная проверка
```bash
npm run lint
```
Выполняет все проверки: TypeScript, ESLint, Stylelint, CSS переменные

### Отдельные проверки
```bash
npm run lint:js          # ESLint
npm run lint:css         # Stylelint
npm run lint:types       # TypeScript проверка типов
npm run check:css-vars   # Проверка CSS переменных
```

## 📊 Данные приложения

### Модель данных кружка
```typescript
interface Club {
  id: string;
  name: string;
  description: string;
  longDescription: string;
  category: 'programming' | 'robotics' | 'security' | 'design' | 'general' | 'sports';
  benefits: string[];
  schedule: { day: string; time: string; room: string }[];
  leader: { name: string; role: string; avatar?: string };
  contacts: { email?: string; telegram?: string; vk?: string };
  image: string;
}
```

### Источники данных
- **Статические данные**: `src/data/clubs.ts`
- **Изображения**: Unsplash API (динамическая загрузка)
- **Иконки**: Lucide React (локальные)
